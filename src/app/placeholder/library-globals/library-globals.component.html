Icons


      
Attributes

      
    
        
  • Supports a string label, positioned centered
    • 
        
  • '-round' adds a border radius to create round icons
    • 
        
  • '-opacity' makes the background transparent to 30%
    • 
      


      
Sizing


      
    
        
  • Default is 24px
    • 
        
  • '-sm' = 12px
    • 
        
  • '-md' = 36px
    • 
        
  • '-lg' = 48px
    • 
      


      
Colors

      
    
        
  • Use any of the SASS color variables
    • 

        
  • 
          Prefix '-color' with a color name modifies the icon fill. For example:
          
      
            
    • -colorSupportingWhite
      • 
            
    • -colorAccentLightRed
      • 
          
    
        
    • 
        
  • -textSupportingWhite modifies the text color to white
    • 
        
  • 
          Background color
          
      
            
    • -backgroundAccentCoolGray
      • 
            
    • -backgroundPrimaryGreen
      • 
            
    • -backgroundAccentRed
      • 
            
    • -backgroundSupportingWhite
      • 
            
    • -backgroundAccentYellow
