_colors.scss


      
Mixin: colorClasses


      

        If any component needs color specific class modifiers. Instead of
        manually writing them or putting it outside of the SCSS with Angular
        data passing: you can use this mixin to quickly generate specific color
        wrapping with CSS classes.
      


      

        There is some maintenance with modifying the array of $colors inside the
        mixin if the color palette changes.
      


      


      

        Generally, you can target the shadow DOM wrapper of ":host" then apply
        the colors to any attribute of a child element.
      


      

        The example below is from the pills component. Color classes are
        generated to target the background-color of the span element that wraps
        the content.

Icon Libary


      

        Icons are all SVG paths using the fill property. They are compiled
        within the 'libary.svg' inside the assets folder.
      


      

        Each icon is a symbol element with an ID. The ideal state is have
        consistent viewBox properties.
      


      

        When importing in new icons, remove any style attributes such as fill or
        stroke.
