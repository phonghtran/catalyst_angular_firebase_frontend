Vertical Nav

The passing array structure accounts for three levels of nesting, and a flag for showing certain items on load.

  • 'active' is the flag to show selected state. Use true if the page being loaded matches to the nav item.
  • 'showOnLoad' is the flag to have the menu open on load. (This is separate from 'active' because some nav items represent on page anchor links while others go to different pages.) Each parent should have this flag match as the loop isn't built to check upwards each time.
